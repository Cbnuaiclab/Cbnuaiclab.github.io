Active Projects

Arm Robot

The Arm Robot Project aims to develop an intelligent bin-picking system that enables robots to detect, localize, and grasp objects in cluttered, unstructured environments. By combining advanced vision techniques such as instance segmentation, graph-transformer reasoning, depth refinement, shape completion, and 6D pose estimation, the project seeks to overcome challenges like occlusion and sensor noise.

Key Technologies: Instance Segmentation, Graph Transformers, 6D Pose Estimation

Status: In Progress

COCOON AI

Integrating Large Language Models (LLMs) with visual data for multimodal AI applications. This project focuses on developing advanced systems that can understand and process both text and images, enabling natural language processing combined with vision-language understanding for comprehensive AI solutions.

Key Technologies: LLMs, Vision-Language Models, Multimodal AI

Status: In Progress

Low-Resource Language NLP

Advancing natural language processing capabilities for low-resource languages, with a focus on Southeast Asian languages. This includes work on tokenization, question answering systems, and language model fine-tuning to bridge the digital divide for underrepresented languages.

Key Technologies: NLP, Tokenization, Fine-tuning, Question Answering

Status: In Progress
